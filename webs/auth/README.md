# Auth Web App

A React app built with Rsbuild, React 19, and Tailwind CSS v4.

## Tech Stack

- **Framework**: React 19 with TypeScript
- **Bundler**: Rsbuild
- **Styling**: Tailwind CSS v4

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Configuration

Create a `.env` file in the root directory:

```env
VITE_API_URL=http://localhost:3001
```
